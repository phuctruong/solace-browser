```bash
cd solace-browser
ls PRIMEMERMAID*.md
```

### Step 2: Understand
```bash
# Read summary (15 min)
cat PRIMEMERMAID_SCOUT_FIXES_SUMMARY.md | head -200

cat PRIMEMERMAID_BEFORE_AFTER_EXAMPLE.md | head -300
```

### Step 3: Create new node
```bash
# Copy template
cp primewiki/PRIMEMERMAID_TEMPLATE_FIXED.md \
   primewiki/mysite-portals.primemermaid.md

# Edit
nano primewiki/mysite-portals.primemermaid.md
```

### Step 4: Fill in data
```bash
# Navigate to site
python persistent_browser_server.py &
curl -X POST http://localhost:9222/navigate \
  -d '{"url": "https://mysite.com"}'

# Test selectors
curl http://localhost:9222/html-clean | jq -r '.html' > /tmp/page.html
grep -o 'class="[^"]*product[^"]*"' /tmp/page.html

# Update template with actual data
nano primewiki/mysite-portals.primemermaid.md
```

### Step 5: Verify
```bash
# Check template is complete
grep -c "Strength:" primewiki/mysite-portals.primemermaid.md
# Should show 5-10+ portals

# Check color scheme
grep "fill:#2563EB\|fill:#10B981\|fill:#DC2626" \
  primewiki/mysite-portals.primemermaid.md
# Should have only 4 colors

# Verify version
grep "^version:" primewiki/mysite-portals.primemermaid.md
```

### Step 6: Commit
```bash
git add primewiki/mysite-portals.primemermaid.md
git commit -m "docs(primewiki): MyWebsite portals with all 11 fixes"
```
