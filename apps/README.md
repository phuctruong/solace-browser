# Solace Browser Starter Bundle

This directory contains the shallow local bundle that ships with Solace Browser.

Included on purpose:
- `manifest.yaml`
- `recipe.json`
- `budget.json`
- `inbox/northstar.md`

Excluded on purpose:
- `requirements/`
- `inbox/tasks/`
- `inbox/notebooks/`
- managed prompts, uplifts, and premium cloud-only context
- outbox evidence

Current starter bundle:
- `solace-yinyang`
- `gmail-inbox-triage`
- `gmail-spam-cleaner`
- `github-issue-triage`
- `calendar-brief`
- `slack-triage`

These are native free starter apps for the Browser/Yinyang local experience.
Apps remain installable in the signed-in browser experience. Paid value lives in
managed reasoning, cloud orchestration, trade-secret uplifts, sync, and remote
control services on Solace AGI.
