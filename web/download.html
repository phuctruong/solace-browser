Installation Instructions

🍎 macOS Installation ▼
  1. Download the correct DMG for your Mac (Apple Silicon: arm64, Intel: x86_64).
  2. Open the DMG file. Drag Solace Browser.app to Applications.
  3. First launch: Right-click the app → Open (needed once to bypass Gatekeeper on first run).
  4. The first-launch wizard will configure file access and LLM settings.
  5. Verify the SHA-256 checksum (optional but recommended):
shasum -a 256 ~/Downloads/SolaceBrowser-1.0.0-mac-arm64.dmg

Compare output to the checksum table above.

🐧 Linux Installation ▼

Choose .deb (Debian/Ubuntu) or AppImage (universal).

.deb package (Debian / Ubuntu / Mint):

sudo dpkg -i ~/Downloads/SolaceBrowser-1.0.0-linux-amd64.deb sudo apt-get install -f # fix any dependency issues

AppImage (all distributions):

chmod +x ~/Downloads/SolaceBrowser-1.0.0-linux-amd64.AppImage ~/Downloads/SolaceBrowser-1.0.0-linux-amd64.AppImage

Verify SHA-256:

sha256sum ~/Downloads/SolaceBrowser-1.0.0-linux-amd64.deb
🪟 Windows Installation ▼
  1. Download SolaceBrowser-1.0.0-windows-x64.msi.
  2. Run the MSI installer. Accept the UAC prompt.
  3. If Windows SmartScreen appears, click More info → Run anyway.
  4. Follow the installer wizard. Solace Browser will appear in Start Menu.
  5. The first-launch wizard configures file access and LLM settings.
  6. Verify checksum in PowerShell:
Get-FileHash .\SolaceBrowser-1.0.0-windows-x64.msi -Algorithm SHA256

System Requirements

🍎 macOS
  • macOS 10.15 Catalina or later
  • Apple Silicon or Intel x86_64
  • 4 GB RAM minimum
  • 500 MB disk space
  • Python 3.10+ (bundled)
🐧 Linux
  • Ubuntu 20.04 / Debian 11+
  • amd64 architecture
  • 4 GB RAM minimum
  • 500 MB disk space
  • libwebkit2gtk-4.0
  • Python 3.10+ (bundled)
🪟 Windows
  • Windows 10 (1903+) or 11
  • 64-bit x64 only
  • 4 GB RAM minimum
  • 500 MB disk space
  • WebView2 Runtime (auto-installed)
  • Python 3.10+ (bundled)
