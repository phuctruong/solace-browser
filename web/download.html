Quick start

Up and running in 60 seconds

Terminal 
# 1. Check the install
solace --version

# 2. Launch with the Yinyang assistant
solace --head --yinyang

# 3. Or run headless (server mode)
solace --headless --port 9222
Alternative

Build from Source

Solace Browser is source-available under the Functional Source License (FSL-1.1-Apache-2.0). Audit the code, build locally, contribute improvements.

Terminal 
git clone https://github.com/solaceagi/solace-browser.git
cd solace-browser
pip install -e .
playwright install chromium
python solace_browser_server.py
