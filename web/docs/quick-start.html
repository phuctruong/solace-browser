Quick Start

1. Install

Solace Browser is a local desktop app — no cloud dependency required. It runs a browser automation server (solace_browser_server.py) and a local web UI (web/server.py).

Option A — Download binary (recommended)

Visit solaceagi.com/browser and download the latest stable build for your OS. Run the installer.

Option B — Run from source

git clone https://github.com/solaceagi/solace-browser
cd solace-browser
pip install -r requirements.txt
python3 solace_browser_server.py --port 9222 &
python3 web/server.py --port 8791

Then open http://localhost:8791/start in your browser.

Option C — One command (npx)

npx solace-browser

This starts both servers and opens the UI automatically. Requires Node.js 18+.

2. Launch

After install, two servers are running:

  • Browser API server — localhost:9222 — Playwright-powered, handles navigate/click/screenshot
  • Web UI server — localhost:8791 — The YinYang interface, app store, settings

Sign in at localhost:8791/start with your Google, GitHub, or email account. This links your Solace AGI account and stores your API key locally.

3. Your first automation

Go to App Store and click Gmail Inbox Triage. The app will:

  • Navigate to Gmail in the controlled browser
  • Read your 20 most recent unread emails
  • Generate a priority-sorted digest
  • Show you the proposed actions for approval

Nothing happens until you approve. This is the core invariant.

4. The approval flow

Every automation pauses at the OAuth3 confirmation gate before taking any action:

1. App proposes: "Archive 12 newsletters, flag 3 urgent items"
2. YinYang shows: scopes requested, budget remaining, estimated cost
3. You click [Approve] or [Reject]
4. Only after [Approve] does any action execute
5. All actions are sealed to your evidence vault with SHA-256

If you leave the page or close the browser, the action is cancelled (fail-closed). There is no auto-approve.

Budget enforcement

Each app has per-action budgets (e.g. Gmail: max 10 sends/day, max 3 deletes/day). Budget gates run before every action. If a budget is exceeded, the action is blocked — not retried, not queued.

5. The YinYang tutorial

On your first visit to Solace Browser, the YinYang tutorial popup walks you through the 5 core concepts in about 2 minutes. You can skip it anytime (press Escape) or re-run it from Settings.

The tutorial supports 13 languages. Click the globe icon 🌐 in the tutorial header to switch languages. Your language choice is saved locally.

To reset the tutorial: localStorage.removeItem('sb_tutorial_v1') in the browser console, then reload.
