Execution Model

LLM called ONCE at preview. Execution is deterministic CPU replay of sealed output.

Preview cost ~$0.003 per run (L2 classification + generation)
Replay cost ~$0.001 per replay (no LLM, CPU only)
Cooldown 5 seconds (Safety B — visible, reputation-impacting)

Lifecycle

  • 1 Trigger — Schedule or manual from Yinyang rail
  • 2 Budget check — B1-B5 fail-closed gates
  • 3 Preview — LLM generates output, shown in bottom rail
  • 4 Approve — User clicks approve (30s timeout = deny)
  • 5 Seal — Output sealed to outbox (chmod 444)
  • 6 Execute — CPU replays sealed output on mail.google.com
  • 7 Evidence — Hash chain sealed, budget decremented
