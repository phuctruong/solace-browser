Typography

Three font stacks: Manrope for display, IBM Plex Sans for body, IBM Plex Mono for code.

Section copy for descriptive text under headings. Kept readable at max-width 60ch.

.section-copy — muted, max-width: 60ch.

Tech Stack

Architecture mirrors solaceagi.com: one CSS file, one JS file, Python server, zero CDN dependencies.

CSS

Single stylesheet

web/css/site.css — 100+ tokens, all components. No CSS-in-JS, no Tailwind, no preprocessor. SHA256 hash for cache busting.

JavaScript

Vanilla JS

web/js/solace.js — IIFE with page-specific init. Particles, reveal observer, hamburger, nav highlight. No framework.

Server

Python servers

web/server.py — dev server with slug routing. solace_browser_server.py — aiohttp CDP server with security middleware.
