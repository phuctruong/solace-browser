Privacy

Local-first. All data stays on your machine unless you opt into cloud sync.

  • History stored locally — ~/.solace/history/
  • Vault encrypted — AES-256-GCM with PBKDF2
  • Tokens in memory only — Never written to disk in browser
  • Cloud sync optional — Evidence + settings sync opt-in
