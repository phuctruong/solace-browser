5 Ways to Control Solace Browser
01
AI Agent
Use with Claude Code, Cursor, or any AI coding agent loaded with Stillwater skills. The agent dispatches tasks directly via MCP or HTTP.
02
Command Line
solace-browser run 'task' — direct CLI execution. Fully scriptable, cron-compatible, pipe-friendly.
03
OAuth3 Web
Grant scopes from solaceagi.com dashboard for remote control. Each scope is explicit, revocable, and audit-logged.
04
Tunnel
Built-in ngrok-like tunnel — expose your local browser to your cloud twin. Connect from anywhere, no port forwarding needed.
05
Download
Get Solace Browser at www.solaceagi.com/browser. Free and open source. Cloud features require a www.solaceagi.com account.
