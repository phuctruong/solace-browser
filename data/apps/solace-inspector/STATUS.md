## Architecture (Agent-Native ‚Äî CRITICAL)

```
CORRECT MODEL:
  Claude Code reads report ‚Üí applies its OWN intelligence ‚Üí analysis is in-session

  run_solace_inspector.py
    Step 1-5: Pure data collection (navigate, ARIA, DOM, heuristics, screenshot)
    Step 6:   llm_analyze ‚Üí RETURNS structured prompt + raw data (NO API call)
    Step 7:   compute_qa_score ‚Üí score from heuristics only (no LLM needed)
    Step 8:   seal_report ‚Üí SHA-256 sealed outbox/report-*.json

  The "persona" is a prompt template injected into Claude Code's analysis, not OpenRouter.

  Runner calls OpenRouter/Together.ai for LLM analysis (adds cost, latency, drift)
```
